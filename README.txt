Após baixar o Unity e o projeto, alguns packages precisam ser instalados no projeto.
Com a Unity aberta execute os seguintes passos.

1- File > Build Settings > Selecionar Android > Switch Platform.

2- Window > Package Manager > Canto Superior Esquerdo selecionar Unity Registry > Adicionar os seguintes pacotes:
   - Text Mesh Pro
   - XR Plugin Manager
   - XR Interaction Toolkit
   - XR Hands
   - Universal RP
	
3- Window > Package Manager > Canto Superior Esquerdo selecionar Unity Registry > Universal RP	

4- Reabrir a Unity.

Após isso o projeto está adequado para receber alterações da forma correta.
